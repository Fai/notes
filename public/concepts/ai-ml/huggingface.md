# Standard Pipeline

Explore
- Hugging Face Hub
Prepare
- Download Model : Transformers
- Download Data : Datasets
Training
- Single GPU LLM : unsloth เอา TRL มาทำให้ใช้ง่าย
- Multi GPU LLM : TRL ต้องรองรับถึงจะใช้ได้
- Any model : Accelerate, PEFT
- Tiny model : Simple Transformers
- Custom Train : PyTorch
Prediction
- Default : Tranformers
- Optimization : Optimum
- API : vLLM
- On-Device : LLaMa C++
Prototype
- App : Gradio
- Deployment : Spaces
Evaluation
