## Cloud Computing Concept

### Service Models
Cloud Offering แต่ละโมเดล ต่างกันที่ใครจะดูแลอะไร 
- Traditional Approach (On Promises) ดูแลเองทุกอย่าง
- Infrastructure as a Services (IaaS) อุปกรณ์ Hardware และ Networking ดูแลโดย Vendor ที่เหลือเราดูแลเอง เหมาะสำหรับคนที่เป็น System Administrators
- Platform as a Services (PaaS) ใช้ Platform จาก Vendor เราดูแลแค่ Application และ Data เหทาะสำหรับ Developers
- Software a a Service (SaaS) ทุกอย่างดูแลโดย Vendor เราเป็นแค่ผู้ใช้ เหมาะสำหรับ End Users
เปรียบเทียบเหมือนเราทำพิซซ่าเองที่บ้านตั้งแต่นวดแป้ง ซื้อแป้งสำเร็จรูปมาทำที่บ้าน สั่งเดลิเวอร์รี่ ไปจนถึงไปสั่งทานที่ร้าน
